- months ≥ 9：`"超过9个月的应急金，安全边际很足，大部分突发情况都能从容应对。"`

#### 中分（12-15分）

```
📊 紧急预备金：{score}分（满分20）

当前覆盖{months}个月支出，距离6个月目标还差{gap}个月（约{gapAmount}）。
```

**计算逻辑**：

- gapAmount = (6 - months) × 月支出
- monthlyAmount = gapAmount ÷ 6（6个月达标）
- timeline = gap ≤ 3 ? "3-4个月" : "6个月左右"


**计算逻辑**：

```
- 应急金达标后，把这部分钱全部用于还债
- 每月{debtPayment}还债，{repayTime}能还清

这样做的好处：
- 先建立应急金，避免意外事件让债务问题恶化
- 应急金达标后全力还债，效率更高
- 6个月后，债务大幅减少，应急金也有了
```

- yearsToRetire = 退休年龄 - age

## 📝 模板设计原则

### 1. 数据驱动
所有模板都用真实数据替换占位符，不使用模糊表述。

### 2. 分段清晰
总体 → 问题 → 行动 → 专属，逻辑递进。

### 3. 语气统一
专业 + 温暖 + 鼓励，避免机械化。

### 4. 可执行性强
给出的建议都能立即落地，有具体金额和时间。

### 5. 给希望
低分段必须包含鼓励语，避免用户焦虑。
